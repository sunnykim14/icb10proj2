# 가격 전략 시사점

<br>

### 도서정가제가 만드는 구조

```
정가 설정 ──→ 고정 10% 할인 ──→ 판매가 결정
     │                                  │
     ▼                                  ▼
가격 경쟁 불가                   콘텐츠·브랜드·
(할인율 동일)                    타이밍이 유일한
                                 차별화 수단
```

<br>

- 더 비싸도 팔리는 도서 = **브랜드(출판사) + 저자 신뢰 + 신간 선점**
- 할인이 안 되면 **번들·사은품·적립금**으로 부가가치 제공
- 정가 30,000원 이상 도서가 TOP 50에 집중 → **고가 전략이 유효**

# SWOT 분석

<br>

| | **기회 (Opportunity)** | **위협 (Threat)** |
|---|---|---|
| **강점 (Strength)** | AI 코딩 도서 붐에 맞춰 **초기 시장 선점** 가능 | AI 도구 변화 속도가 **도서 생명주기보다 빠름** (6~12개월) |
| **약점 (Weakness)** | **하반기 신간 공백**으로 매출 편중 발생 | 수험서 시장에 **이지선생님 등 과점 저자** 존재 |

<br>

**SO 전략** : 지금 당장 AI 신간 선점 — 골든타임 극대화
**ST 전략** : 빠른 리뉴얼 주기 계획, 시리즈화로 생명주기 연장
**WO 전략** : 하반기 AI 심화서·커뮤니티 콘텐츠로 갭 보완
**WT 전략** : 자체 수험서 저자 네트워크 구축, 독점 콘텐츠 확보

# IT 도서 시장의 골든타임

<br>

> **"IT 도서 시장은 지금 'AI 코딩 도구 교과서'의 골든타임이다."**

<br>

2026년 상반기 전체 베스트셀러의 **48.8%** 가 신간으로 집중됐고, 그 중심에 Claude Code · Vibe Coding이 자리 잡고 있다.

AI 도구의 생명주기는 **6~12개월** — 지금이 최고 매출 시점이며, 이 창은 빠르게 닫힌다.

<br>

서점이 지금 당장 해야 할 일은 **3가지**:

1. **Speed** — AI 신간은 출간 후 2주 이내 발주·진열이 매출을 결정
2. **Segment** — AI 직장인과 자격증 수험생은 메시지와 채널을 분리
3. **Data Loop** — 베스트셀러 자동 수집 → 예측형 운영으로 전환
